新建Macdown的.md文件自带名称、时间的方法

定位到目标文件夹下，使用如下命令：

hexo new xxx(指文件名)

P.S.写这个博客时，发现了两个快速移动光标位置的快捷键：

command + ↑ ↓ ← →(指方向键的上、下、左、右键)

Alt + ← →(指方向键的左、右键)

**请自行体会不同之处**
